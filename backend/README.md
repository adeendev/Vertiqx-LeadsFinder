# Vertiqx Leads Backend

FastAPI backend for the Vertiqx lead generation system.

## Environment Variables

Add these to your Hugging Face Space Settings → Variables and secrets:
- `HOSTINGER_PASS`
- `APIFY_API_TOKEN`
- (Optional) `GMAIL_PASS`, `GMAIL_USER`, etc.
